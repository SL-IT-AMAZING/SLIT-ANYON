## 주요 인사이트 요약

### 1️⃣ 시장 기회는 명확하다
- 한국 기업의 **72%가 AI 도입**에 관심 있지만, **보안/규제 때문에 망설임**
- AI PoC의 **88%가 프로덕션 실패** → 온프레미스 AI 솔루션 절실
- 노코드 시장은 연평균 **27.4% 성장** 중
- 개발자 **2만 명 부족** → 노코드 수요 폭발

### 2️⃣ 타겟 고객은 명확하다
- **금융/공공/제조/의료** 등 보안이 중요한 산업
- **망분리 환경**에서 AI를 써야 하는 기업
- **레거시 시스템**과 연동이 필요한 기업
- **개발자 부족**으로 고민 중인 IT 부서

### 3️⃣ 핵심 차별점은 명확하다
- **온프레미스 우선 설계** → 망분리에서도 AI 사용 가능
- **AI 네이티브 통합** → 개발 시간 80% 단축
- **한국 기업 특화** → SSO, 감사로그, 규제 준수
- **경쟁사 대비 유일** → 온프레미스 + AI의 조합

### 4️⃣ 영업 전략은 명확하다
- **문제 중심** 접근: "AI 쓰고 싶은데 보안 때문에 못 쓰시죠?"
- **데모 중심** 설득: 말보다 보여주기
- **점진적 확장**: PoC → 파일럿 → 전사 확산
- **챔피언 찾기**: 내부 옹호자를 만들어라

### 5️⃣ 성공의 핵심은 실행이다
- **빠른 후속 조치** (24시간 내 답변)
- **맞춤형 접근** (획일적 피치 금지)
- **경청과 학습** (고객이 선생님)
- **관계 구축** (거래가 아닌 파트너십)

## 자주 묻는 질문 (FAQ)

### Q1. 비기술자도 이 자료를 이해할 수 있나요?
**A**: 네! `03_필수개념정리.md`는 비전공자도 이해할 수 있도록 쉽게 설명되어 있습니다. 기술 질문이 나오면 `06_치트시트.md`의 대응 카드를 참고하세요.

### Q2. 가장 먼저 읽어야 할 파일은?
**A**: `06_치트시트.md`를 먼저 읽으세요. 5분이면 전체 맥락을 파악할 수 있습니다.

### Q3. 타겟 기업을 어떻게 선정하나요?
**A**: `01_타겟기업리스트.md`에서 산업별/규모별로 우선순위가 정리되어 있습니다. 망분리 환경을 쓰는 기업부터 시작하세요.

### Q4. 기술 질문을 받으면 어떻게 대응하나요?
**A**: `06_치트시트.md`의 "핵심 대응 카드"를 외우세요. 모르는 질문은 "기술팀 확인 후 24시간 내 회신"이 정답입니다.

### Q5. 경쟁사와 어떻게 차별화하나요?
**A**: `04_영업전략가이드.md`에 경쟁사 분석과 차별화 전략이 자세히 나와 있습니다. 핵심은 "온프레미스 + AI"입니다.
