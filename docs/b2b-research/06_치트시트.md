## 핵심 대응 카드 (외울 것)

### 1. **망분리 환경에서 작동하나요?**

✅ "완전 망분리 환경에서 작동합니다. 외부 API 의존성이 전혀 없고, 모든 AI 처리를 온프레미스에서 합니다."

### 2. **SSO 연동 가능한가요?**

✅ "SAML 2.0과 OIDC를 네이티브로 지원합니다. LDAP/AD 연동도 당연히 됩니다."

### 3. **보안/규제 준수는?**

✅ "TLS 1.3 암호화, AES-256 데이터 암호화, 전체 감사 로그 제공합니다. 개인정보보호법과 금융/공공 규제를 준수합니다."

### 4. **온프레미스 배포는 어떻게?**

✅ "Docker 컨테이너로 제공되며, Kubernetes 환경 지원합니다. 설치부터 운영까지 가이드 제공합니다."

### 5. **AI는 어떻게 작동하나요?**

✅ "Ollama 기반 온프레미스 LLM과 RAG를 사용합니다. OpenAI 같은 외부 API 호출이 전혀 없습니다."

### 6. **경쟁사 대비 차별점은?**

✅ "온프레미스 우선 설계 + AI 네이티브 통합 + 한국 기업 특화 기능입니다. 망분리에서도 AI를 쓸 수 있는 유일한 솔루션입니다."

### 7. **모르는 질문이 나오면?**

✅ "정확한 답변을 위해 기술팀과 확인 후 24시간 내로 회신 드리겠습니다."

## 30초 엘리베이터 피치

### 🎯 CTO / 개발팀장용

"애니온은 온프레미스에서 작동하는 AI 기반 노코드 플랫폼입니다. 망분리 환경에서도 ChatGPT 수준의 AI를 사용해 내부 업무 도구를 개발자 없이 빠르게 만들 수 있습니다. 보안 걱정 없이 AI를 도입하고 싶은 기업을 위한 솔루션입니다."

### 💼 IT팀장 / 실무자용

"애니온을 쓰면 외주나 개발자 없이도 사내 업무 시스템을 만들 수 있습니다. AI가 자동으로 워크플로우를 설계하고, 드래그앤드롭으로 화면을 구성하면 끝입니다. 온프레미스라 보안 걱정도 없고, 기존 시스템과 연동도 쉽습니다."

### 👔 경영진 / 의사결정자용

"한국 기업들은 AI를 도입하고 싶지만 보안과 규제 때문에 못하고 있습니다. 애니온은 망분리 환경에서도 AI를 안전하게 사용할 수 있게 해주는 플랫폼입니다. 개발 비용을 70% 줄이고, 시스템 구축 기간을 1/5로 단축시킵니다."

## 절대 하면 안 되는 것 ⚠️

### ❌ OpenAI API를 쓴다고 하지 마세요

→ 망분리에서 작동 안 한다는 인상을 줍니다. "온프레미스 LLM"이라고 말하세요.

### ❌ "나중에 개발하겠습니다" 하지 마세요

→ 현재 없는 기능은 "로드맵에 있으며 우선순위 조정 가능합니다"라고 하세요.

### ❌ "저는 비기술자라서" 하지 마세요

→ 기술 질문은 "기술팀과 확인 후 정확히 답변드리겠습니다"로 대응하세요.

### ❌ 가격을 먼저 말하지 마세요

→ 요구사항을 충분히 듣고, 가치를 설명한 후에 가격을 제시하세요.

### ❌ 경쟁사를 깎아내리지 마세요

→ "저희는 이런 점이 강점입니다" 방식으로 차별점을 설명하세요.
